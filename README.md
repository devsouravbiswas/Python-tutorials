# Python Tutorials — Basics to Advanced

A structured collection of Python tutorials, example code, and small projects — from absolute basics up to advanced topics. Each lesson contains a short README, runnable examples, and optional tests.

## Goals
- Teach Python progressively (Beginner → Intermediate → Advanced).
- Provide clear explanations + runnable code.
- Encourage contributions (exercises, improvements, translations).

## Repo layout
- `basics/` — beginner lessons (variables, control flow, functions)
- `intermediate/` — data structures, OOP, modules
- `advanced/` — concurrency, type hints, design patterns, performance
- `examples/` — small real-world projects and scripts
- `tests/` — pytest tests for examples where applicable
- `docs/` — optional documentation

## Getting started (local)
```bash
# clone
git clone https://github.com/<your-username>/python-tutorials.git
cd python-tutorials

# recommended: create venv
python -m venv .venv
source .venv/bin/activate    # Linux / macOS
.venv\Scripts\activate       # Windows
